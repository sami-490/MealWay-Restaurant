## 🌟 About The Project

**MealWay Restaurant** is a beautifully designed, fully responsive web application tailored for the culinary industry. Built with a mobile-first approach, it provides a seamless user experience across desktops, tablets, and smartphones. Whether you want to showcase your menu, promote upcoming events, or allow customers to book a table, this template handles it all elegantly.

## ✨ Key Features

- **📱 Fully Responsive:** Flawless layout adaptation from large desktop monitors down to mobile screens.
- **🎨 Modern UI/UX:** Clean, appetizing design with smooth scrolling and engaging hover animations.
- **📅 Table Booking System:** Integrated form structure for reserving tables.
- **🍔 Interactive Menu:** Filterable and beautifully laid out restaurant menu showcasing dishes and prices.
- **👨‍🍳 Chef Showcase:** Dedicated section to highlight the culinary experts behind the food.
- **📸 Image Gallery:** Glightbox integration for a stunning fullscreen image viewing experience.
- **💬 Testimonials:** Swiper.js integration for a smooth, touch-friendly customer review carousel.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites

You only need a modern web browser to view the site, but to run it locally as a developer, you can use any basic HTTP server (e.g., `http-server`, `Live Server` in VS Code, or Python's `http.server`).

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sami-490/MealWay-Restaurant.git
   ```
2. **Navigate into the project directory:**
   ```bash
   cd MealWay-Restaurant
   ```
3. **Open the project:**
   Simply double-click `index.html` to open it in your browser, or start a local server:
   ```bash
   npx http-server
   ```
   Then visit `http://localhost:8080` in your browser.
